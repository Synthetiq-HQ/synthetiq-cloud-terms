# Synthetiq Cloud Terms

This repository contains the Terms of Service for Synthetiq Cloud, the cloud sync and backup service used by the Synthetiq Player app.

## What is Synthetiq Cloud?

Synthetiq Cloud lets you sync your favourites, watch history, reading progress, settings, and preferences across all your devices.
